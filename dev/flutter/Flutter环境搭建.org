#+TITLE: Flutter开发环境的搭建


** 把.pub-cache/bin加入PATH

- windows :: 
#+BEGIN_SRC 
C:\Users\<your-username>\AppData\Roaming\Pub\Cache\bin
#+END_SRC

- mac或linux :: 
#+BEGIN_SRC 
$~/.pub-cache/bin
#+END_SRC
